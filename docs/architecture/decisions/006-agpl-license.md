# ADR-006: AGPL-3.0 License

## Status

Accepted

## Context

Yappie is an open-source project developed as a TFM (Master's Thesis). We need a license that encourages open contributions while protecting against proprietary forks.

## Decision

License the project under AGPL-3.0.

## Consequences

- **Positive:** Strong copyleft ensures modifications remain open source
- **Positive:** Network use clause covers SaaS deployments
- **Negative:** May deter corporate contributions due to copyleft requirements
- **Negative:** Incompatible with some permissively licensed dependencies
